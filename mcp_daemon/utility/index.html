Utility functions, macros, and types

This module provides utility functions, macros, and types for working with the MCP protocol. Utility functions and macros for MCP daemon implementation.

This module contains utility functions and macros that are used throughout the MCP daemon implementation. The primary functionality provided is the #[server] macro for implementing the Daemon trait, which greatly simplifies the creation of MCP-compliant servers.
