MCP client

The Client struct provides methods for interacting with an MCP server, including initializing the connection, listing and accessing resources, calling tools, and more.

To create a Client, use the with_server method or ClientBuilder. The method to create a Client performs an initialize request to the server and returns control when the request completes.

§Example

ⓘ
use mcp_daemon::client::Client;
use mcp_daemon::schema::*;
use mcp_daemon::server::{Server, DefaultServer};

struct MyServer;

impl Server for MyServer {
    // Implement required methods
    // ...
}

let server = MyServer;
let client = Client::with_server(server).await?;

// Get prompt list
let prompts = client.prompts_list(None).await?;

// Get resource list
let resources = client.resources_list(None).await?;

// Call tool
let params = CallToolRequestParams {
    name: "tool_name".to_string(),
    arguments: Some(serde_json::Map::new()),
};
let result = client.tools_call(params).await?;

Client for interacting with MCP servers.

The Client struct provides methods for interacting with an MCP server, including initializing the connection, listing and retrieving prompts, accessing resources, and calling tools.

It handles the communication details of the Model Context Protocol and provides a high-level API for working with MCP servers.

§Example

ⓘ
use mcp_daemon::client::Client;
use mcp_daemon::schema::*;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Create a client that connects to a server at the specified URL
    let client = Client::new("http://localhost:8080");

    // Initialize the client with the server
    let init_result = client.initialize().await?;
    println!("Connected to server: {}", init_result.server_info.name);

    // List available prompts
    let prompts = client.prompts_list(None).await?;
    for prompt in prompts.prompts {
        println!("Found prompt: {}", prompt.name);
    }

    // Call a tool
    let tool_result = client.tools_call(CallToolRequestParams {
        name: "example_tool".to_string(),
        arguments: serde_json::Map::new(),
    }).await?;

    Ok(())
}
